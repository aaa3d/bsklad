  frAdvPictureView is the add-in component for FastReport 2.4x
(www.fastreport.ru). It can work with pictures in the following formats: 
gif, jpeg, bmp, ico, emf, wmf.

  The component is based on the TAdvPicture component by TMS Sofware
(www.tmssoftware.com). You must download it and install before installing
the frAdvPictureView component.

  frAdvPictureView has the same properties and methods as standard
TfrPictureView.
